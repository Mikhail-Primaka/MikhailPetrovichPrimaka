Форматы файлов.

.Doc или .doc (от англ.document) - расширение имени файла, используемое для файлов,
представляющих текст, с разметкой или без. Расширение .Dos часто используется для обозначения
простых текстовых файлов без форматирования, однако позже стало использоваться для двоичных
форматов с разметкой.
Документ MS Word
Portable Document Format (PDF) - межплатформенный формат электронных документов,
разработанный фирмой Adobe Systems с использованием рядо возможностеей языка PostScript.
ДокументPDF
ZIP - популярный формат сжатия данных и архивации файлов. Файл в этом формате обычно имеет
расширение zip и хранится в сжатом или несжатом виде один или несколько файлов, которые нужно
из него извлечь путем распаковки с помощью специально программы.
